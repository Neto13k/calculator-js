# 🧮 Calculator JS

Uma calculadora web moderna com histórico de cálculos, desenvolvida com **HTML, CSS e JavaScript puro** para praticar manipulação do DOM, lógica de programação e estilização avançada.

🔗 **[Acesse o projeto ao vivo](https://neto13k.github.io/calculator-js/)**

---

## ✨ Funcionalidades

- ➕ Operações básicas: adição, subtração, multiplicação e divisão
- 🕒 Histórico dos últimos **5 cálculos** com horário registrado
- 🔁 Clique em qualquer entrada do histórico para reutilizar a expressão ou o resultado
- 🧹 Botão **C** para limpar o visor
- ⚠️ Tratamento de erros para expressões inválidas
- 📱 Layout **responsivo** para mobile e desktop

## 🚀 Como Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/neto13k/calculator-js.git
   ```

2. Acesse a pasta:
   ```bash
   cd calculator-js
   ```

3. Abra o arquivo `Calculadora.html` no seu navegador — sem necessidade de servidor ou dependências.

## 📚 O que Aprendi

- Manipulação dinâmica do DOM com JavaScript puro
- Uso de `querySelectorAll` e `forEach` para eventos em múltiplos elementos
- Tratamento de erros com `try/catch`
- Criação e inserção de elementos HTML via JavaScript (`createElement`, `appendChild`)
- Construção de layouts com CSS Grid/Flexbox e variáveis CSS
- Deploy automático com **GitHub Actions** e **GitHub Pages**
